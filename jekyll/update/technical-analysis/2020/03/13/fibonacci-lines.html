Fibonnaci Lines

Mar 13, 2020

If Fibonacci Line drops back to slightly above 0.618 you can expect the next climb to be longer than the first climb.

If Fibonacci Line drops back to slightly above 0.5 you can expect a climb that is about the equal length with the first one.

If Fibonacci Line drops back to slighly below 0.5 but above 0.382, then anything goes, usually it will lead to a reversal of fortune.

If Fibonacci Line drops back to slighly above 0.236 or below 0.236 you can expect a reversal of fortune. That means the next climb will be less than 0.5 (around 0.236 or 0.382). We also call this the reversal drop.

If the reversal drops to between 0.786 to 0.618. The follow up drop will be about the equal length or longer than the first drop.

If the reversal drops to slightly above 0.5 you can expect the follow up drop to be about the equal length with the first drop.
